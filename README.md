# FLORA — Цветочный магазин

Next.js 16 + Tailwind + PostgreSQL

## Запуск

1. Установите зависимости:
```bash
npm install
```

2. Создайте `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

3. Укажите `DATABASE_URL` в `.env` (PostgreSQL)

4. Выполните миграции:
```bash
npx prisma migrate dev
```

5. Запустите dev-сервер:
```bash
npm run dev
```

## Функционал

- Главная: баннер + каталог популярных букетов
- Каталог с фильтрами и сортировкой
- Квиз подбора букета
- Карточка товара (количество, в корзину)
- Корзина с апсейлами (открытка)
- Оформление заказа (телефон с маской, тоггл открытки)
- Вход/выход (demo: любой email)
- Личный кабинет: профиль, история заказов, мои поводы
- История заказов сохраняется в PostgreSQL
